
Static GitHub Pages dashboard for tracking the Monster Versus City Destruction thesis framework.

## How to use
1. Upload this folder to a GitHub repository.
2. Enable GitHub Pages from the repository settings.
3. Edit `data/daily-plan.json` as each day is completed.
4. Add Obsidian logs into `/logs` and worksheet PDFs into `/worksheets`.

The spreadsheet can be exported to CSV/JSON later and used to update `data/daily-plan.json`.
