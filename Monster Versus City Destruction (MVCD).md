
Monster Versus City Destruction is a modular Unreal Engine 5 C++ framework focused on scalable destruction systems for large-scale urban environments.

The project explores:
- Real-time destruction systems
- Chaos physics integration
- Event-driven damage systems
- Structural integrity simulation
- Multiplayer destruction synchronization
- Performance optimization for large-scale destruction

This repository also contains:
- Daily development logs
- Technical documentation
- Research notes
- Benchmarks
- Architecture diagrams
- Dissertation support material

---

# Tech Stack

- Unreal Engine 5.7.4
- C++
- Chaos Destruction
- GitHub Pages
- Obsidian
- Markdown Documentation

---

# Project Goals

The main objective is to research and implement a destruction framework capable of supporting immersive large-scale destruction scenarios while remaining performant and multiplayer-compatible.

Inspired by:
- Battlefield
- The Finals
- Modern large-scale physics systems
